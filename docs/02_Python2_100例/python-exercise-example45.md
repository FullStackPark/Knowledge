Python 练习实例45
=============

 [Python 100例](python-100-examples.md)


 **题目：** 统计 1 到 100 之和。

 **程序分析：** 无

 程序源代码：

  实例
--

 <pre>

#!/usr/bin/python
# -*- coding: UTF-8 -*-
tmp = 0
for i in range(1,101):
    tmp += i
print 'The sum is %d' % tmp
</pre>

  以上实例输出结果为：


```

The sum is 5050

```
