Python 字典(Dictionary) values()方法
================================

  描述
--

 Python 字典(Dictionary) values() 函数以列表返回字典中的所有值。

 语法
--

 values()方法语法：

 
```

dict.values()

```

 参数
--

  * NA。
  返回值
---

 返回字典中的所有值。

 实例
--

 以下实例展示了 values()函数的使用方法：

 
```

#!/usr/bin/python

dict = {'Name': 'Zara', 'Age': 7}

print "Value : %s" %  dict.values()

```

 以上实例输出结果为：

 
```

Value : [7, 'Zara']

```
