Python id() 函数
==============

 [![Python 内置函数](../images/up.gif)
 Python 内置函数](python-built-in-functions.html)


  描述
--

 **id()** 函数用于获取对象的内存地址。

 语法
--

 id 语法：

 
```

id([object])

```

  参数说明：

  * object -- 对象。
  返回值
---

 返回对象的内存地址。

 实例
--

 以下实例展示了 id 的使用方法：

  <pre>

>>>a = 'runoob'
>>> id(a)
4531887632
>>> b = 1
>>> id(b)
140588731085608
</pre>
