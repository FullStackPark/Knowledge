 **题目：**列表转换为字典。 

 **程序分析：**无。

 程序源代码：

  实例
--

 <pre>

#!/usr/bin/python
# -*- coding: UTF-8 -*-
i = ['a', 'b']
l = [1, 2]
print dict([i,l])
</pre>

 以上实例输出结果为：

 
```

{'a': 'b', 1: 2}

```
