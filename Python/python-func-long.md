Python long() 函数
================

 [![Python 内置函数](../images/up.gif)
 Python 内置函数](python-built-in-functions.html)


  描述
--

 **long()** 函数将数字或字符串转换为一个长整型。

 语法
--

 long() 函数语法：

 
```

class long(x, base=10)

```

 参数
--

  * x -- 字符串或数字。
 * base -- 可选，进制数，默认十进制。
  返回值
---

 返回长整型数。

 实例
--

 以下实例展示了 long() 的使用方法：

  <pre>

>>>long()
0L
>>> long(1)
1L
>>> long('123')
123L
>>> 
>>>
</pre>
