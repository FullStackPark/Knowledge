Python os.pathconf() 方法
=======================

 [![Python File(文件) 方法](../images/up.gif)
 Python OS 文件/目录方法](os-file-methods.html)


  ### 概述

 os.pathconf() 方法用于返回一个打开的文件的系统配置信息。

 Unix 平台下可用。

 ### 语法

 **fpathconf()** 方法语法格式如下：


```

os.fpathconf(fd, name)

```

 ### 参数

  * **name** -- 文件描述符
